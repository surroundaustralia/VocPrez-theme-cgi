Modelling & Tools

Vocabulary Modelling

CGI conceives of vocabularies as being synonymous with the Simple Knowledge Organization System (SKOS)'s skos:ConceptScheme class, so each CGI vocab is an instance of skos:ConceptScheme. All the individual terms within vocabs - instances of skos:Concept - are also presented in a flat list per vocabulary: a skos:Collection.

Profiles

The content of all vocabs in this system can be accessed individually in web page (HTML) and machine-readable forms. Each list of vocabs, vocabulary and Concept web page contains a link to the listing of all profiles and formats available for that object. Look for the Alternate Profiles hyperlink.
